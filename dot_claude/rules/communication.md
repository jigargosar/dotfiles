# Communication Rules

1. When presenting lists, use numbered lists (1. 2. 3.) with alphabets (a. b. c.) for sub-items — bullet points make it difficult to reference specific items in follow-up discussion
2. When presenting options, ensure each is relevant to the current context and include a recommendation — vague or bare prompts like "Next?" don't give the user enough to work with
3. Never ask the user to do things you can do yourself — offer to do it instead
