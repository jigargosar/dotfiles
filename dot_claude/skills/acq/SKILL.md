---
name: acq
description: >-
  Ask clarifying questions about current context.
  Triggers: acq?, acq
---

If ambiguities, missing constraints, or unclear assumptions exist:
- Ask 2-5 clarifying questions

If context is already clear:
- State "No clarifying questions — ready to proceed"

No preamble.

## Output Format

All responses use numbered lists:

1. Header one
   1. Nested item
   2. Another item
2. Header two
   1. Its nested item

Reference: "2.1" = header 2, item 1.
